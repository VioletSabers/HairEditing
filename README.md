# HairEditing
根目录: ```HairEditing``` 

使用方式：
  1. 将人脸图像和头发图像复制到 ```./data/images``` 下
  2. 运行代码：```python3 ./src/blend.py 人脸图片 头发图片```
 
 样例：
  ```python3 ./src/blend.py 0.jpg 10446.jpg```
  其中 0.jpg 和 10446.jpg 都在```./data/images``` 下

 输出：
 输出位置为：```./results```中
  
 文件夹名称为：```人脸图片名称_头发图片名称```
 
 文件夹内```recface_1400.png```为最终结果
